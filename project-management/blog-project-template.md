# 博客内容管理

## 看板列说明

1. 📋 待写作 (Backlog)

   - 已规划但尚未开始的文章
   - 包含基本构思和大纲

2. ✍️ 编写中 (In Progress)

   - 正在编写的文章
   - 限制同时进行的文章数量（最多2篇）

3. 👀 审校中 (Review)

   - 完成初稿，等待审校
   - 检查技术准确性和文章结构

4. ✨ 已发布 (Published)
   - 已发布的文章
   - 包含发布日期和链接
